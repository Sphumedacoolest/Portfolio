how it works

The API

Open Trivia DB
A completely free, open trivia API with thousands of verified questions across 24 categories
HTML Entity Decoding
The API returns HTML-encoded strings (e.g. &amp;). The app decodes them using a hidden textarea trick — a classic JS pattern.
Fisher-Yates Shuffle
Correct and incorrect answers are merged then shuffled using the Fisher-Yates algorithm so the right answer never appears in the same slot.

Core Logic

trivia.js — fetch questions JS
// Build URL from user config async function fetchQuestions(amount, cat, diff) { let url = `https://opentdb.com/api.php` + `?amount=${amount}&type=multiple`; if (cat) url += `&category=${cat}`; if (diff) url += `&difficulty=${diff}`; const res = await fetch(url); const data = await res.json(); return data.results; }
trivia.js — shuffle answers JS
// Merge & shuffle so answer order // is random every question function shuffleAnswers(q) { const answers = [ ...q.incorrect_answers, q.correct_answer ]; // Fisher-Yates shuffle for (let i = answers.length - 1; i > 0; i--) { const j = Math.floor(Math.random() * (i+1)); [answers[i], answers[j]] = [answers[j], answers[i]]; } return answers; }

Features

📡
Uses API
Fetches live questions from Open Trivia DB
⏱️
Countdown Timer
15 seconds per question with a live progress bar. Auto-submits and penalises on timeout.
🎛️
Configurable
Choose from 14 categories, 3 difficulty levels, and 4 question counts before each game.
🔀
Shuffled Answers
Answers are merged and shuffled with Fisher-Yates so the correct answer never has a fixed position.

Tracks score, correct answers, wrong answers, and remaining time in real time during the quiz.

Results Screen
End-of-game breakdown with final score, accuracy percentage, and a one-click replay button.
